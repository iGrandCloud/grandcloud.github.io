用 Travis CI 自动部署Hexo博客到 GitHub

前言

使用hexo静态博客已经半年多了，总体感觉挺好用的。但是有一点一直让我很苦恼，由于这是静态博客，所以每次写完博客都需要先generate,再发布deploy。而且博客的基本配置信息没办法同时更新到github，还需要我再push上去，这样一番下来，虽然花费不了太长时间，但是时间长了就比较难以忍受了。
